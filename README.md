# AI Projects Portfolio using LangChain and Ollama

A collection of AI applications powered by **LangChain**, **Ollama**, and **Mistral LLM**, running locally without relying on external AI APIs.

## Projects Included

### 1. AI Chatbot with Memory

A context-aware chatbot that remembers previous interactions using LangChain's ChatMessageHistory.

#### Features

* Local LLM inference using Ollama
* Mistral LLM integration
* Conversational memory
* Context-aware responses
* CLI and Streamlit interfaces

---

### 2. AI Voice Assistant

A voice-enabled assistant capable of listening to user queries, generating responses using Mistral, and speaking them back.

#### Features

* Speech-to-Text using SpeechRecognition
* Text-to-Speech using pyttsx3
* Conversation memory
* Voice interaction
* Offline execution

---

### 3. AI-Powered Web Scraper & Summarizer

A web scraping application that extracts website content and generates concise AI-powered summaries using Mistral.

#### Features

* Website scraping using Requests and BeautifulSoup
* AI-powered content summarization
* Streamlit web interface
* Local inference using Ollama
* Real-time webpage analysis
