# AI Chatbot using LangChain and Ollama

A simple AI chatbot powered by Mistral running locally through Ollama.

## Features

- Local LLM inference
- LangChain integration
- Interactive command-line chatbot
- Offline execution

## Tech Stack

- Python
- LangChain
- Ollama
- Mistral

## Installation

```bash
pip install langchain langchain-community langchain-ollama
```

## Run

```bash
python basic_ai_agent.py
```
